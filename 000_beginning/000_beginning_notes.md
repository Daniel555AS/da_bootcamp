## What is MySQL?

*MySQL* es un Sistema de Gestión de Bases de Datos Relaciones (Relational Database Management System - RDBMS).

Es un software que: [It is a software that:]

- Guarda datos [Stores data].
- Permite administrarlos [Allows data management].
- Ejecuta instrucciones SQL [Executes SQL instructions].
- Es muy usado en apps web [Is widely used in web applications].
- Organizar la información en tablas (filas y columnas) [Organizes information into tables (rows and columns)].

*MySQL* es una implementación específica y fue creado inicialmente por *MySQL AB*, sin embargo, actualmente es propiedad de *Oracle Corporation*.

[*MySQL* is a specific implementation and was initially created by *MySQL AB*; however, it is currently owned by *Oracle Corporation*.]
